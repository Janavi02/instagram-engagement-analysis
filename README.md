# 📊 Instagram Engagement Analysis

## 📌 Overview
This project analyzes Instagram post engagement data to identify key factors influencing performance.

## 🎯 Objective
To find the best posting time, day, and content type for maximum engagement.

## 🛠️ Tools Used
- MySQL
- Tableau Public
- Excel

## 📂 Dataset
Instagram Analytics dataset from Kaggle

## 🔍 Analysis
- Analyzed best posting hour using SQL
- Identified best day for engagement
- Compared media types performance
- Built Tableau dashboard

## 📊 Key Insights
- Hour 3 has highest engagement
- Sunday performs best
- Image posts get more likes
